##  About the Project

SkillXChange is a full-stack web platform built for SRM AP students to exchange skills instead of money.
It helps students connect with peers who can teach what they want to learn, creating a collaborative campus learning ecosystem.

---

##  Team

We are a team of **4 developers** working on this project as part of a university hackathon.

---

##  Tech Stack

### Frontend

* React
* Tailwind CSS

### Backend

* Node.js
* Express

---

##  Features

* SRM AP email-restricted signup/login
* Student profiles with teach & learn skills
* Smart skill matching dashboard
* Connection requests system
* Real-time chat & call support
* Interactive campus map

---

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/skillxchange.git
cd skillxchange
```

### 2. Install dependencies

#### Backend

```bash
cd server
npm install
```

#### Frontend

```bash
cd client
npm install
```

### 3. Run the app

#### Backend

```bash
npm run dev
```

#### Frontend

```bash
npm run dev
```
